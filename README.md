# Analyse-de-biodiversite
Projet universitaire de 4è année en génie géomatique

ScriptGEE : Permet de faire rouler le script sur la plateforme de GEE

ScriptGEE_Entrainement : Le script qui a été utilisé pour sortir le endmember utilisé sur les années passées, futures et sur différents emplacements

ScriptGEE+python_auto : C'est le script qui route sur le serveur et appel l'API de GEE et du Cloud pour obtenir la classification automatiquement
