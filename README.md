# Analyse-de-biodiversite
Projet universitaire de 4è année en génie géomatique

ScriptGEE : Permet de faire rouler le script sur la plateforme de GEE

ScriptGEE_Entrainement : C'est le script qui a été utilisé pour sortir les endmembers utilisés sur d'autres périodes temporelles et sur d'autres campus

ScriptGEE+python_auto : C'est le script qui roule sur le serveur et appel l'API de GEE et du Cloud pour obtenir la classification automatiquement
